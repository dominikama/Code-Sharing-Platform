REST API

There are 3 REST endpoints available:

  • GET: api/code/{uuid} - allows you to see a code snippet by typing it's unique token
  • GET: api/code/latest - allows you to see 5 latest code snippets without time and view restrictions
  • POST: api/code/new - allows you to post a new code snippet

All data should be send and will be received in JSON format.

WEB

There are 5 web endpoints available:

  • /code/{uuid} - allows you to see a code snippet by typing it's unique token
  • /- allows you to see a home page with 5 latest code snippets
  • /code/new - allows you to post a new code snippet
  • /getAll - allows you to see all code snippets without time and view restrictions
  • /more - this is a link to this page
