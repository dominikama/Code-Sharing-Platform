About the project

Welcome to the code sharing platform. Here you can post new snippets. There are some strict rules: To post a snippet, click "Add new snippet". You'll be asked to fill in the form. You can set time, and view restriction - if they are triggered your code snippet will be deleted from the database. If you don't want your snippet to have any restrictions you have to type 0 in those places. To view a single code snippet you've got to know its unique token and type it in the URL. There are two possible formats in which you can access and send data: HTML or JSON. To get more information about REST API and Web interface visit the tabs above.
